# **App Name**: Mahendra Design Studio

## Core Features:

- Dynamic Content Display: Render content for hero, about, work experience, skills, and testimonials sections, ensuring all details are clearly presented.
- Interactive Project Gallery: Showcase portfolio projects in a responsive grid, featuring thumbnail images, titles, descriptions, tags, and 'View Details' buttons for each entry.
- Contact Inquiry System: Provide a contact form for visitors to send inquiries, along with Mahendra's email, phone, and social media links (LinkedIn, Behance, Dribbble).
- AI Project Description Tool: Offer a generative AI tool to assist Mahendra in creating compelling, concise descriptions for new portfolio projects based on input keywords or brief outlines.
- Responsive Layout & Navigation: Implement a mobile-first, fully responsive design with a sticky navigation bar at the top for easy site traversal and smooth scrolling.
- Theme Toggle: Allow users to switch between a clean dark mode and a crisp light mode for optimal viewing preference.
- Subtle UI Feedback: Integrate smooth hover effects, micro-interactions, and tasteful animations to enhance user engagement and visual appeal.

## Style Guidelines:

- Primary Color: A vibrant violet (#6952E0) that evokes creativity and professionalism, contrasting elegantly with the dark background.
- Background Color: A sophisticated, desaturated dark violet (#17161D) providing a clean, modern canvas.
- Accent Color: A clear, bright blue (#759FF0) used for calls-to-action, highlights, and interactive elements, providing strong contrast.
- Headline Font: 'Poppins' (sans-serif) for its modern, geometric aesthetic, conveying precision and style. Body Font: 'Inter' (sans-serif) for excellent readability and a neutral, objective feel in longer text blocks.
- Utilize a consistent set of minimalist, vector-based icons that complement the clean, modern design across all sections and for social media links.
- Implement a full-width, section-based layout with a strong emphasis on content hierarchy. Portfolio projects will use a dynamic grid system. Ensure optimal presentation across all device sizes with a mobile-first approach.
- Incorporate smooth, subtle transition animations for section scrolling, navigation hover states, project card interactions, and the light/dark mode toggle to enhance the user experience without distraction.
